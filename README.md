# Portfólio React

Portfólio profissional desenvolvido em React.

## Páginas

- Home
- Projetos
- Contato

## Componentes

- Header
- Footer
- ProjectCard

## Instalação

git clone https://github.com/seuusuario/portfolio-react

cd portfolio-react

npm install

npm run dev

## Deploy

Pode ser feito com Vercel ou GitHub Pages.
